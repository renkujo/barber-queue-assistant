# Generated Asset Pack V1 — Pastel Paper Queue

Raster PNG assets generated with Codex gpt-5.5 image generation after rejecting the first hand-authored SVG draft.

These assets are the current preferred visual asset direction for Barber Queue Assistant.

## Files

- `app-icon-pastel.png` — app icon / brand mark direction
- `barber-chair-cutout.png` — barber chair illustration for empty/status states
- `barber-pole-cutout.png` — barber/shop cue
- `queue-ticket-cutout.png` — blank queue ticket motif
- `scissors-comb-cutout.png` — service cue
- `line-notification-cutout.png` — optional notification cue, no LINE logo/text
- `no-show-warning-cutout.png` — late/no-show warning cue
- `walk-in-queue-cutout.png` — walk-in/today queue cue

## Important QA note

The files are PNG raster assets, but the generated outputs do **not** currently include a real alpha channel. They should be treated as source/reference raster assets on warm cream backgrounds for now.

If the final app needs compositing on arbitrary backgrounds, next step is either:

1. regenerate with stronger transparent-output support, or
2. create cleaned transparent cutouts from these PNGs with a proper background-removal workflow.

## Usage

Use these for:

- visual direction;
- prototype illustrations;
- brand/asset refinement;
- early UI mockups on the same Pastel Paper Queue cream/paper backgrounds.

Do not treat them as final production transparent cutouts yet.
